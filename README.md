
# Credit Card Fraud Detection

This project demonstrates a **credit card fraud detection system** using Python.

## Project Overview

- Implements machine learning techniques to identify fraudulent credit card transactions.
- Includes data preprocessing, model training, and evaluation.
- Focuses on demonstrating coding and machine learning workflow.

## How to Run

1. **Install dependencies**:

```bash
pip install -r requirements.txt
````

2. **Run the main script**:

```bash
python credit_fraud_detect.py
```

> Note: The full dataset is not included due to size constraints. This project demonstrates the workflow and techniques used for fraud detection.
